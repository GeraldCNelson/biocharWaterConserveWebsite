# Digestibility Metrics (NIR Clean Data — Set 4)

## What Set 4 is for
Set 4 focuses on variables that more directly describe **digestibility** and **fiber utilization**, often important for:
- predicting intake and animal performance,
- understanding forage quality changes with maturity,
- and connecting plant structure with energy availability.

## Typical variables (Set 4)
Common candidates:
- **NDFD 48 hr (Dry Basis)** (Neutral Detergent Fiber Digestibility)
- **IVTDMD 48 hr (Dry Basis)** (In Vitro True Digestible Dry Matter)
- **Lignin (Dry Basis, %)**

## Notes on interpretation
- Digestibility metrics can be very informative but sometimes have inconsistent availability across labs/years.
- **Lignin** tends to increase with maturity and reduces fiber digestibility.

## Why this is a separate set
Digestibility variables are more specialized and can be interpreted incorrectly without context.
Separating them keeps Set 1 clean while allowing deeper analysis when needed.
