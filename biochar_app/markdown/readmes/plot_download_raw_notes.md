Plot Download Notes - Raw
-------------------------

Contents

- Raw downloads contain logger time-series data for the selected plot configuration.
- When the plot is grouped by depth, the download includes all sensor depths for the selected strip and logger location.
- When the plot is grouped by logger location, the download includes Top, Middle, and Bottom logger locations for the selected strip and depth.

Column naming notes

- Raw sensor columns describe individual strip, logger-location, and sensor-depth measurements.
- Sensor depth codes appear after the variable name.
- Logger location codes appear near the end of logger-derived column names.
