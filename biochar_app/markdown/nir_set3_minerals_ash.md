# Minerals & Ash (NIR Clean Data — Set 3)

## What Set 3 is for
Set 3 tracks **mineral content** and **ash**. This is useful for:
- linking soil chemistry and amendments to plant mineral uptake,
- diagnosing nutrient imbalances,
- and understanding forage mineral supply (especially for grazing livestock).

## Typical variables (Set 3)
Common candidates:
- **Ash (Dry Basis, %)**
- **Calcium (Ca, Dry Basis, %)**
- **Phosphorus (P, Dry Basis, %)**
- **Potassium (K, Dry Basis, %)**
- **Magnesium (Mg, Dry Basis, %)**

## Notes on interpretation
- **Ash** is a broad measure and can be influenced by contamination (soil/dust) as well as plant mineral status.
- Minerals can vary with growth stage and irrigation timing.
- For treatment comparisons, consistency in sampling and handling matters a lot.

## Why this is a separate set
Minerals are important but can distract from the “core forage quality” story.
Keeping them separate prevents Set 1 from becoming cluttered while still making minerals easy to access.
