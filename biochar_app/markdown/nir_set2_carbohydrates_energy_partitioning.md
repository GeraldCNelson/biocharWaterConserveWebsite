# Carbohydrates & Energy Partitioning (NIR Clean Data — Set 2)

## What Set 2 is for
Set 2 focuses on **non-structural carbohydrates** that influence:
- rapid energy availability,
- rumen fermentation dynamics,
- and plant stress/seasonal physiology.

This set is useful when you want to understand **energy sources beyond fiber** and how they shift with season, irrigation, temperature, or soil amendments.

## Typical variables (Set 2)
Common candidates (depending on what Ward reports consistently):
- **Non-Fiber Carbohydrates (NFC, Dry Basis, %)**
- **Starch (Dry Basis, %)**
- **Water-Soluble Carbohydrates (WSC, Dry Basis, %)**
- **Ethanol-Soluble Carbohydrates (ESC, Dry Basis, %)**
- **Fructans (Dry Basis, %)**

## Notes on interpretation
- **Starch / soluble carbs** often respond strongly to maturity and growth conditions.
- These values can be more sensitive to harvest timing than fiber metrics.
- If a variable has spotty coverage across dates/years, it may be moved to a later “exploratory” set.

## Why this is a separate set
Carbohydrates answer a different question than Set 1:
- Set 1 is “overall forage quality (classic diagnostics)”
- Set 2 is “where is the energy coming from?”
